Features

A more deliberate way to work with AI.

Local-first runtime

The desktop shell talks to a local agent runtime, keeping execution, memory, and session state close to your machine.

Visible reasoning and tools

Watch streamed text, reasoning blocks, tool calls, and tool results unfold in real time instead of guessing what happened.

Workspace-scoped sessions

Keep conversations grounded in the project you are actually working on, with persistent sessions that remain easy to revisit.

Semantic memory

Retrieve useful context before each turn and preserve durable project knowledge as work continues.

Multi-provider model support

Use OpenAI, Anthropic, compatible APIs, and more while keeping the desktop experience coherent.

Native-feeling macOS design

Cometline is shaped like a serious desktop tool: calm surfaces, keyboard-first flow, and UI motion that respects focus.

How it works

Three layers, each with a clean responsibility.

01

Cometline

The Electron and Svelte desktop shell handles the interface, settings, animations, and OS integration.

02

CometMind

The local agent runtime orchestrates multi-step runs, persistence, tools, memory, and streaming API behavior.

03

Comet SDK

The provider boundary normalizes model I/O, streaming, retries, and tool-call assembly across vendors.
